# Health Systems Analytics Dashboard

## Project Overview
An interactive Power BI solution designed to track health system performance across multiple indicators. 

## Key Features
- **Mobile-Optimized:** Specialized layout for field health officers.
- **Advanced DAX:** Custom measures for patient outcomes and wait times.
- **Dynamic Filtering:** Slicers for region, date, and facility type.
